# mean-crud
node + angular with mongodb crud

use npm install in both the folders to install dependencies ..

To start node Server : <br>
cd nodeJs <br>
node ./bin/www <br>
started at :http://localhost:500/

<br>
To start Angular app : <br>
cd angularJs <br>
ng serve -o <br>
started at :http://localhost:4200/
